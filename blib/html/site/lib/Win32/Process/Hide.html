NAME

Win32::Process::Hide - Perl extension for hiding your process.

SYNOPSIS

  use Win32::Process::Hide;
  HideProcess();

DESCRIPTION

This module allow you to hide your process, to prevent some negligent user terminate your process. PLEASE USE IT IN RIGHT WAYS

EXPORT

None by default.
